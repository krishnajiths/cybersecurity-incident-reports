# Cybersecurity Incident Reports

This repository documents real-world cybersecurity incidents in a structured format. Each report includes the date, target, sector, attack type, threat actor, attack method, impact, response, current status, lessons learned, and severity rating.

## Purpose

The goal of this project is to practice cybersecurity analysis, improve incident response understanding, and build a public portfolio of security research.

## Topics Covered

- Supply chain attacks
- Ransomware
- Zero-day exploits
- Threat actors
- Incident response
- Lessons learned from real attacks
